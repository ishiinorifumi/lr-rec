# LR-REC

ListenRadio（リスラジ） の放送局を聴いたり、予約録音（MP3）できる非公式の Windows アプリです。※当アプリ及び当ページは、ListenRadioとそのサービス及び株式会社ディーピーエヌとは一切関係がありません。

バージョン: 1.1.1


## ダウンロード

[**LR-REC-1.1.1.zip をダウンロード**](https://github.com/ishiinorifumi/lr-rec/releases/tag/v1.1.1)
（Releases ページの Assets から `LR-REC-1.1.1.zip` を取得してください）

## インストール

1. 配布 ZIP を好きな場所に解凍します（インストール不要・管理者権限不要）。
2. `LR-REC.exe` をダブルクリックで起動します。

> 必要に応じて、`LR-REC.exe` を右クリック →「スタートメニューにピン留め」や、
> ショートカットを作成してスタートアップに置くと便利です。

> 日本国内からのみ聴取・録音できます（配信側の地域制限）。

## 使い方

1. 左の一覧から放送局を選びます（上の「局一覧を更新」で最新の一覧を取得。地域や名前で絞り込み、★でお気に入り登録）。
2. 「▶ 再生」で聴く（局名・ロゴ・現在の番組名・放送時間帯が表示されます）。音量はスライダーで調整。
3. 予約録音は「追加」から作成します。日時・録音時間・繰り返し（1回／毎日／毎週）・品質（64/128/256kbps）・保存先・ファイル名を指定できます。
4. 予約時刻になると自動で録音されます（アプリ起動中のみ。開始に失敗しても予約時間内は自動で録り直します）。
5. ウィンドウの × でタスクトレイに常駐します。終了はトレイアイコンを右クリック →「終了」。

## ファイル名の指定

ファイル名はテンプレートで指定できます。未指定（既定）は次の形式です。

```
{番組名}-{yyyymmdd}-{hhmm}
```

予約の編集画面では「変数を挿入 ▾」から変数を入れられ、入力欄の下に実際のファイル名の例が表示されます。

使える変数:

| 変数 | 意味 |
|---|---|
| `{番組名}` | 録音開始時の番組名（取得できなければ局名） |
| `{局名}` | 放送局名 |
| `{ラベル}` | 予約に付けた任意ラベル |
| `{yyyymmdd}` | 日付（例: 20260615） |
| `{hhmm}` | 時刻（例: 0725） |
| `{yyyy}` `{MM}` `{dd}` `{HH}` `{mm}` | 年/月/日/時/分 |
| `{channelid}` | チャンネルID |

- ファイル名に使えない文字（`\ / : * ? " < > |`）は自動で全角等に置き換えられます。
- 録音は MP3 で保存され、タイトル・局名・日付のタグが付きます。

## 保存先

既定の保存先は「ミュージック\LR-REC」です。アプリ上部の「保存先」や、各予約で個別に変更できます。
