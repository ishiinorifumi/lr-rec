# LR-REC

ListenRadio（リスラジ） の放送局を聴いたり、予約録音（MP3／M4A）できる非公式の Windows アプリです。
中学生がAIで作成しています。
※当アプリ及び当ページは、ListenRadioとそのサービス及び株式会社ディーピーエヌとは一切関係がありません。

バージョン: 1.2.0

## 更新履歴

### 1.2.0
- M4A（無劣化）録音に対応しました。配信されている音声をそのまま、再エンコードせずに保存します（音質の劣化なし）。予約録音・今すぐ録音で「MP3」「M4A 無劣化」を選べます。

### 1.1.2
- 「毎週」予約で、予約を作成した曜日に関係なく初回が当日（や直近）の指定時刻に録音されてしまう不具合を修正。初回から正しく次の指定曜日に録音します。
- 「毎日」予約で、当日の指定時刻を過ぎてから作成した場合に誤って「録音失敗（時間切れ）」と表示されることがあった点を修正（翌日以降の最初の時刻に合わせます）。
- 「1回のみ」予約で過去の日時を指定できてしまう点を修正（未来の日時のみ指定可能に）。
- 別局を同じ時間帯に録音するなどでファイル名が重複した場合、後から録音したファイルで上書きされていた不具合を修正。重複時は末尾に ` (1)`, ` (2)`… を付けて別ファイルとして保存します。
- 再生開始時のログ出力を取りやめ（再生・停止はログに残りません。録音関連のログはこれまで通り）。

## インストール

1. 配布 ZIP を好きな場所に解凍します（インストール不要・管理者権限不要）。
2. `LR-REC.exe` をダブルクリックで起動します。

> 必要に応じて、`LR-REC.exe` を右クリック →「スタートメニューにピン留め」や、
> ショートカットを作成してスタートアップに置くと便利です。

> 日本国内からのみ聴取・録音できます（配信側の地域制限）。

## 使い方

1. 左の一覧から放送局を選びます（上の「局一覧を更新」で最新の一覧を取得。地域や名前で絞り込み、★でお気に入り登録）。
2. 「▶ 再生」で聴く（局名・ロゴ・現在の番組名・放送時間帯が表示されます）。音量はスライダーで調整。
3. 予約録音は「追加」から作成します。日時・録音時間・繰り返し（1回／毎日／毎週）・形式（MP3／M4A 無劣化）・品質（MP3 は 64/128/256kbps、M4A 無劣化は配信そのまま）・保存先・ファイル名を指定できます。
4. 予約時刻になると自動で録音されます（アプリ起動中のみ。開始に失敗しても予約時間内は自動で録り直します）。
5. ウィンドウの × でタスクトレイに常駐します。終了はトレイアイコンを右クリック →「終了」。

## ファイル名の指定

ファイル名はテンプレートで指定できます。未指定（既定）は次の形式です。

```
{番組名}-{yyyymmdd}-{hhmm}
```

予約の編集画面では「変数を挿入 ▾」から変数を入れられ、入力欄の下に実際のファイル名の例が表示されます。

使える変数:

| 変数 | 意味 |
|---|---|
| `{番組名}` | 録音開始時の番組名（取得できなければ局名） |
| `{局名}` | 放送局名 |
| `{ラベル}` | 予約に付けた任意ラベル |
| `{yyyymmdd}` | 日付（例: 20260615） |
| `{hhmm}` | 時刻（例: 0725） |
| `{yyyy}` `{MM}` `{dd}` `{HH}` `{mm}` | 年/月/日/時/分 |
| `{channelid}` | チャンネルID |

- ファイル名に使えない文字（`\ / : * ? " < > |`）は自動で全角等に置き換えられます。
- 録音は MP3、または M4A（無劣化コピー）で保存でき、タイトル・局名・日付のタグが付きます。M4A 無劣化は配信中の音声を再エンコードせずそのまま保存します。

## 保存先

既定の保存先は「ミュージック\LR-REC」です。アプリ上部の「保存先」や、各予約で個別に変更できます。
